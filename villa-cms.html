Workflow Guide

Follow these steps to update your website content and deploy to GitHub Pages.

How to Update Your Website

1

Edit Content

Use this CMS to update text content in all languages and upload images.

2

Export Files

Download content.json and all images with proper file names.

3

Update Repository

Place files in your project folder and push to GitHub.

4

Live!

GitHub Pages will automatically update your live website.

File Structure: Your project should have this structure:
